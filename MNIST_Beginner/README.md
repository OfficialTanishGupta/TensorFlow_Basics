# MNIST Digit Classifier (TensorFlow)

This project demonstrates a beginner-level TensorFlow neural network
to classify handwritten digits using the MNIST dataset.

## Tech Stack
- TensorFlow
- Keras
- Google Colab

## Accuracy
~98% on test data

## Features
- Visualized MNIST dataset
- Plotted training/validation accuracy & loss
- Visualized correct and incorrect predictions
- Saved trained model

## Results
Test accuracy ~98%
